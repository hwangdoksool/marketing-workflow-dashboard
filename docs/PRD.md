# PRD: 마케팅 워크플로우 대시보드

> **버전:** v1.0
> **작성일:** 2026-04-01
> **작성자:** 황독술 🦅
> **상태:** 초안 — 황성우님 리뷰 대기

---

## 1. 배경

### 현재 상태
- **기존 대시보드** (`rf-marketing-dashboard.pages.dev`): 칸반 보드 + 주간 리포트 자동 생성
  - tasks.json 기반 태스크 관리 (칸반/리스트 뷰)
  - reports.json 기반 주간 리포트 (채널별 성과 요약)
  - 담당자/카테고리 필터, 우선순위 표시
- **3.31 미팅 진단**: 8주간 실행 밀도는 높았으나, 가설→실험→학습 사이클 부재로 소통 비효율 반복
- **운영 가이드 확정** (4월~): 가설 기반 업무, Creative Brief, 주간 리뷰 4부 구조

### 왜 새로 만드는가
기존 대시보드는 **"뭘 하고 있는가"** (칸반)에 초점. 새 대시보드는 **"왜 하는가 → 결과가 어떤가 → 다음에 뭘 할 건가"** (워크플로우)에 초점.

단순 리디자인이 아니라 **데이터 파이프라인 + 의사결정 도구**로의 전환.

---

## 2. 목표

1. **주간 리뷰 자동화**: 데이터 수집→정리→리포트를 대시보드가 수행, 마케터는 "읽고 판단"만
2. **구매자 역추적**: 실구매 → GA4 행동 → 유입 채널 → 체험 DB 연결, 건별 풀 여정 시각화
3. **Brief 등록/검증**: 스키마 9요소 기반 Brief 작성 UI → 가설 등록 → 결과 입력 → 학습 축적
4. **최종: AI 에이전틱 워크플로우**: Brief 제안, 이상 신호 감지, 가설 자동 검증

## 4. 탭 구조

### 탭 1: 📊 주간 리뷰 (Weekly Review)

운영 가이드의 4부 구조를 그대로 구현:

**0부: 구매자 역추적**
- 이번 주 구매 건별: 유입 채널 / 방문 횟수 / 경유 페이지 / 체험 여부
- 누적 패턴: 구매자 페르소나별 비중 변화
- 데이터 소스: 아임웹 주문 → GA4 사용자 매칭 → Meta 전환 → 체험 DB

**1부: 채널별 성과**
- Meta Ads: 캠페인별 Spend/CTR/CPC/전환, 소재별 피로도
- GA4: 세션/유입채널/전환퍼널/이탈 구간
- 아임웹: 실주문 vs GA4 전환 수 대조
- 인스타: Reach/Saves/Shares, 콘텐츠별
- 네이버 SA: 광고그룹별 CTR/CPC/전환
- B2B 리드: 신규/발송/응답/미팅 전환

**2부: 가설 검증**
- 진행 중 Brief 목록 → 가설 → 결과 (맞음/틀림/미확인)
- 틀린 가설: 어디서 틀렸는가 (타겟/메시지/포맷/채널)
- 예상 못 한 발견

**3부: 이번 주 계획**
- 인사이트 기반 다음 실험 가설 + Brief 초안

**4부: 이슈/블로커**
- 리소스 병목, 소재 소진, 기술 이슈, 의사결정 필요 사항

### 탭 2: 📋 Brief 관리 (Brief Registry)

Creative Brief 라이프사이클 관리:

- **Brief 등록**: 스키마 9요소 기반 폼 (페르소나/JTBD/VP/소재구성/채널/가설/검증기준)
- **상태 흐름**: 작성 → 협의 중 → 제작 중 → 라이브 → 검증 완료
- **가설 보드**: 가설별 상태 (검증 중/유효/무효/보류)
- **학습 아카이브**: 검증 완료된 가설들의 축적 → 패턴 발견

### 탭 3: 🎯 태스크 (Tasks) — 기존 칸반 계승

기존 rf-marketing-dashboard의 칸반/리스트 뷰 마이그레이션:
- 상태(예정/진행중/피드백대기/완료) × 우선순위(P1/P2/P3)
- 담당자/카테고리 필터
- Brief 연결 (Brief → Task → 결과)

### 탭 4: 📈 트렌드 (Trends)

시계열 데이터 시각화:
- 주간 주요 지표 추이 (Spend/CPC/전환/매출)
- 채널별 효율 비교
- 소재 피로도 곡선
- 구매자 유입 채널 비중 변화

## 8. 첫 번째 리뷰 = 2~3월 전체 회고 (상세)

운영 가이드에 명시된 "첫 번째 리뷰":

> "8주간의 데이터를 전부 정리하여 종합 리뷰. 캠페인 타임라인, 소재별 성과, 구매자 전수 역추적, 채널별 효율 추이, 학습 정리."

**구체적으로:**
1. 캠페인 타임라인: 2/3~3/31 모든 캠페인 시작/종료/예산 변경 시점
2. 소재별 성과: 각 크리에이티브의 CTR/CPC/전환 비교
3. 구매자 전수 역추적: 2~3월 모든 구매자의 유입→전환 여정
4. 채널별 효율 추이: 주차별 Meta/GA4/네이버 비교
5. 학습 정리: 무엇이 작동했고, 무엇이 안 됐는가

이 회고가 대시보드의 **첫 콘텐츠**이자, 주간 리뷰 자동화의 **기준선**이 된다.
