## Executive Summary

### 핵심 숫자

### 가장 큰 발견 3가지

1. **view_item → begin_checkout 구간이 최대 병목** — 매주 4,000~10,000건의 상품 조회 대비 결제 시작은 46~137건(전환율 0.8~2.4%). 가격 정당화와 결제 UX가 핵심 개선점.
2. **스포엑스(W12) 효과는 트래픽에 집중, 매출 전환은 제한적** — W12에 세션이 12,147(전주 대비 +70%)로 급등했으나 실매출은 ₩796만(본체 2대)으로 W11과 동일. 광고비만 ₩137.5만으로 최고치.
3. **네이버 SA는 최고 효율 채널** — CTR 9.6~16.4%, 이탈률 19.8~38.4%로 전 채널 최저. 소규모 예산(주당 ₩1.2~3.0만)이지만 구매 의도가 높은 유저를 안정적으로 유입.

### 다음 4주 권고사항

1. **퍼널 하단 최적화 우선**: 상품 페이지 → 결제 전환율 개선에 집중. 분할결제, 체험 후기 강조, 가격 앵커링 등 A/B 테스트 권장
2. **네이버 SA 예산 2~3배 확대**: 현재 주당 ₩1.5~3만 수준 → ₩5~10만으로. 이탈률이 가장 낮고 구매 의도가 높은 채널
3. **Meta 소재 피로도 관리**: CPC가 W7 ₩113 → W13 ₩179로 지속 상승. 2~3주 주기 소재 교체 + 25-55 남성 타겟 세분화
4. **GA4 purchase 이벤트 정비**: 아임웹 실구매 대비 GA4 이벤트가 3~5배 과다 — 중복 발화 원인 파악 후 수정 필요

- **Organic Social**: W10에서 급등(456세션, 전주 37 대비 +10.6배). 인스타그램 포스팅 재개 + 커뮤니티 활동 효과 추정.
- **Direct**: 857→1,984로 증가. 브랜드 인지도 상승의 지표. 단, 이탈률 67~74%로 높아 일부는 봇/오분류 가능.
- **Referral**: 소규모(19→81)이지만 꾸준히 증가. 외부 매체 노출 확대.

### 4. 구매자 역추적

#### 4.1 GA4 구매 이벤트의 유입 채널 분포

GA4 weekly 파일의 funnel 데이터에서 채널별 구매 기여도를 직접 추출할 수 없음 (GA4 API에서 funnel×channel 교차 데이터 미수집). 대신 채널별 이탈률과 전환 퍼널을 간접 추론:

**구매 가능성 높은 채널 (이탈률 기준)**:
1. **Paid Search** (이탈률 19.8~38.4%): 검색 의도 기반, 가장 높은 전환 잠재력
2. **Organic Search** (이탈률 25~35%): SEO 유입, 정보 탐색 후 구매 결정
3. **Organic Social** (이탈률 31~37%): 인스타그램/커뮤니티 경유, 관여도 높음
4. **Display/Meta** (이탈률 52~58%): 볼륨은 최대이지만 탐색적 유입 위주

#### 4.2 첫 방문 → 구매 터치포인트 (추정)

기존 누적 분석(alltime-journey-2026q1)에 따르면:
- 재방문 유저 체류시간 219초 vs 신규 53초 → **구매는 대부분 재방문에서 발생**
- 추정 여정: Meta 광고(인지) → 검색/Direct(재방문) → 상품 조회 → 구매
- sessions_per_user ≈ 1.14로 대부분 1회 방문이지만, 구매자는 2~3회 방문 추정

**한계**: 개별 구매자 수준의 채널 어트리뷰션은 GA4 User Explorer 또는 BigQuery 연동 없이 불가. 현재 데이터로는 간접 추론만 가능.

### 6. 8주간 학습 정리

#### 효과가 있었던 것

1. **Meta ASC(Advantage+) 전환 캠페인**
   - 8주간 메인 트래픽 드라이버. 알고리즘 자동 최적화가 수동 타겟보다 효과적.
   - W12 스포엑스 기간에 CTR 7.0%, CPC ₩116으로 최고 효율 달성.

2. **네이버 SA 파워링크**
   - 소규모 예산이지만 CTR 10%+, 이탈률 20~30%대로 최고 품질 채널.
   - 구매 의도가 확실한 유저를 효율적으로 포착.

3. **인스타그램 콘텐츠 (W10~)**
   - 포스팅 재개 시 Organic Social 세션 10배 증가. 소셜 활동의 트래픽 기여 확인.

4. **스포엑스 시기 광고 확대**
   - 이벤트 기간 CPC가 오히려 낮아짐. 시장 관심도가 높은 시기에 광고 집행 효율 극대화.

5. **점진적 광고비 증액 전략**
   - W7 ₩27만 → W12 ₩137만까지 매주 20~50%씩 증액. 급격한 증액 없이 효율 모니터링하며 스케일.

#### 효과가 없었거나 개선 필요한 것

1. **view_item → begin_checkout 퍼널 병목**
   - 전환율 0.8~2.4%. 매주 수천 명이 상품을 보지만 결제를 시작하지 않음.
   - **원인 추정**: 고가(₩388만) 제품 특성상 즉각 구매 어려움, 분할결제 옵션 부재, 경쟁사 비교.

2. **소재 피로도 미관리**
   - CPC W7 ₩113 → W13 ₩179로 58% 상승. 동일 ASC 캠페인 8주 연속 운영.
   - 2~3주 주기 소재 교체 체계 필요.

3. **GA4 이벤트 설정 부정확**
   - purchase 이벤트 3~5배 과다 발화. 데이터 기반 의사결정의 신뢰도 훼손.
   - W12~W13에서 개선 조짐이나 완전 해결 필요.

4. **B2B 캠페인 CPC 과다**
   - ₩867~1,018/클릭. 리드 품질 확인 없이는 효율 판단 불가. 리드→상담→계약 전환 추적 필요.

5. **인스타그램 포스팅 불규칙**
   - W7~W8 포스팅 0건. 소셜 트래픽 기여가 확인되었으므로 일관된 발행 스케줄 필요.

6. **W6~W9 아임웹 매출 미추적**
   - 4주간 매출 데이터 없어 초기 마케팅 효과 정량화 불가. 향후 데이터 수집 자동화 필요.
